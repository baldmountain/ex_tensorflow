# ExTensorflow

**Tensorflow wrapper for elixir**

Excellent nif example code

http://andrealeopardi.com/posts/using-c-from-elixir-with-nifs/

You need to set:

```bash
export ERL_INCLUDE_PATH=/usr/local/Cellar/erlang/20.0/lib/erlang/usr/include
```
